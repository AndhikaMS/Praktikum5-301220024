Membuat dan Mengelola Branch pada Git dan GitHub

Pengenalan

Branch dalam Git digunakan untuk mengerjakan fitur baru atau melakukan percobaan tanpa mengubah kode utama. Dengan GitHub, kolaborasi antar pengembang menjadi lebih mudah melalui sistem branch ini.

Langkah-langkah Dasar

  1. Buat branch baru: git branch nama-branch
  2. Pindah ke branch tersebut: git checkout nama-branch
  3. Lakukan perubahan dan commit seperti biasa
  4. Gabungkan branch ke main: git merge nama-branch
  5. Hapus branch jika sudah tidak diperlukan: git branch -d nama-branch

Manfaat Menggunakan Branch

  • Memisahkan pekerjaan antar fitur
  • Memudahkan kolaborasi tim
  • Mencegah konflik pada kode utama
